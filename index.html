HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 tag should only appear once on each webpage.
  • The footer element is at the bottom of a page and usually contains the author, copyrights, site map and navigation.
  • Elements that are nested inside other elements are called child elements.
  • The element that hosts all the children elements is called a parent element.
  • The body element is an example of sematic HTML element, the name describe it's content.
  • We should aim to use mostly sematic elements
  • Non sematic elements such as div, which creates a division and span which creates an inline division are still used.
  • The meta element defines metadata such as character set, description, keywords, author and viewport.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can add to selectors with a comma allowing the declaration to affect both of them.
  • Text-align changes where a title or heading sits in the webpage, for example centre
  • We need to link the HTML and CSS sections by using the link element.
  • Class attributes allow us to share a CSS rule to any element we choose, by assigning the rule to a class with a class selectior.
  • By adding a . before an attribute makes it so it is designated as a class selector.

Git

  • git status is used to check what branch we are currently on.
  • git checkout -b branch-name is used to create a new branch and it switches to it.
  • git checkout main takes us to the main branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer execute code in a script
  • An if statement is a way of introducing decision making in our code.
  • Code will only be executed if the if statement is true.
  • An array is a single variable to hold a group of related data.
  • A for loop uses the predictable pattern of indices to perform a task on all items in an array.
